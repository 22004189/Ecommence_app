# Kota King 

Kota King is a web application for ordering fast food (Kotas, Burgers, Fries) online. It features a user dashboard for placing orders and tracking history, and an admin dashboard for managing orders and users.

## Features

- **User Authentication**: Sign up and Login using Email/Password or Google.
- **Menu Browsing**: Filterable menu with categories.
- **Cart System**: Add items, adjust quantities, and calculate totals.
- **Order Placement**: Users can place orders with specific notes (Pickup/Delivery).
- **Real-time Tracking**: Users can see their order status (Pending, Preparing, Completed) update in real-time.
- **Admin Dashboard**:
  - View daily revenue and order stats.
  - Manage order statuses.
  - Manage users.

## Technologies Used

- HTML5, CSS3, JavaScript (ES6 Modules)
- Firebase Authentication
- Firebase Firestore (Database)

## How to Run

1. Clone the repository.
2. Open `Main.html` in your browser (or use a local server like Live Server in VS Code).
